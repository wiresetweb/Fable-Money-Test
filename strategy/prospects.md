# Prospect list — cold outreach (Phase 2)

**Goal:** 50+ US home-service businesses with *observable* web pains. **Rule: no row without a specific, observable pain** — "could use a website" doesn't count.

Sourced 2026-06-12 via web search (snippets/URLs only — container egress blocks fetching the sites themselves, so **every pain must be re-verified in a browser before sending**; snippets can be stale).

Status flow: `identified → verified → drafted → sent → replied → call → scoped → paid → delivered`

## Priority scoring
- **A** — severe pain + obvious offer fit (Facebook-only or free-subdomain site; high-ticket trade). Send first.
- **B** — clear pain, decent fit ("call for pricing" only path; dated site).
- **C** — pain present but weaker signal or lower-ticket niche.

## Pain → pitch mapping (template variants in `outreach/templates/cold-first-touch.md`)
| Observed pain | Variant | Ask |
|---|---|---|
| Facebook-only presence | A | Website ($499) |
| Free-subdomain site (wixsite/godaddysites/weebly) | B | Website ($499) |
| "Call for pricing" / phone-only quoting | C | Private quote builder ($350) |
| Dated / broken / under-construction site | D | Website rebuild ($499) |

## Process (Laz + agent)
1. Pick the day's batch (start with all **A**s, ~10/day max — quality beats volume).
2. **Re-verify the pain in a browser** (open their page; confirm it's still true; grab owner first name + best email/contact form — "Hi Mike" beats "Hi there").
3. Personalize the matching variant — replace every bracket, run the honesty checklist.
4. Send from hello@wiresetweb.com; log to `outreach/sent/` (date, business, variant, outcome).
5. One follow-up after 4–5 days. "No thanks" = permanent stop, log it.

## Sourcing recipe notes (for the next round)
- **`site:godaddysites.com [trade]`** — best yield everywhere (~60–80% usable). Bonus signals: numbered subdomains ("…0", "…1"), default slugs.
- **`site:wixsite.com [trade]`** — great hooks (default "/mysite", "/website" slugs); spot-check for mapped real domains.
- **`site:wordpress.com` / `site:weebly.com`** — good but highest false-positive rate (stale leftovers of businesses that upgraded). Verification search per prospect is mandatory.
- **`site:facebook.com [trade] [city]`** — strongest pain when it hits, but only ~20% truly site-less; budget one verification search each.
- **`site:sites.google.com`** — mostly spam; rare real finds (Service 1 HVAC, Nic & Dan's).
- **"under construction" / "copyright 2019"** — zero yield, drop.
- **Highest-value discovered pattern:** *duplicate-indexed free subdomain alongside a custom domain* (Hensley, Uncle Fergie's, Peninsula) — a concrete, verifiable SEO problem you can name in line 1 of the email.
